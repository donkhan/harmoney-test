Getting Stocks for given Branch and Currency

URL : /harmoney2/get-stocks/{branch-name}/{currency}

HTTP Method: GET

Payload: None

Sample Output :
{
"code":0,
"description":"Success",
"data":[
{
"name":"Branch",
"totalStocks":23421.2
},
{
"name":"Tellers",
"totalStocks":12312
}
]
}

Error Code :

0 - Success 1 - Branch Not Present 2 - Currency Not Present

Example : /harmoney2/get-stocks/Trichy/USD

Test Program For reference: MooseGETTest.java, MooseTestThread.java
